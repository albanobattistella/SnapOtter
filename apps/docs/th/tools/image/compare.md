# Image Compare {#image-compare}

อัปโหลดภาพสองภาพเพื่อคำนวณแผนที่ความแตกต่างระดับพิกเซลและเปอร์เซ็นต์ความคล้ายคลึงเชิงตัวเลข เอาต์พุตเป็นภาพความแตกต่างที่เน้นบริเวณที่เปลี่ยนแปลงด้วยสีแดง

## API Endpoint {#api-endpoint}

`POST /api/v1/tools/image/compare`

รับข้อมูลฟอร์ม multipart พร้อมไฟล์ภาพ **สอง** ไฟล์ ไม่ต้องมีฟิลด์การตั้งค่า

## Parameters {#parameters}

เครื่องมือนี้ไม่มีพารามิเตอร์ที่ปรับแต่งได้ อัปโหลดไฟล์ภาพให้ครบสองไฟล์พอดี

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| file (first) | file | Yes | ภาพแรก |
| file (second) | file | Yes | ภาพที่สอง |

## Example Request {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/compare \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@original.jpg" \
  -F "file=@modified.jpg"
```

## Example Response {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "similarity": 94.52,
  "dimensions": { "width": 1920, "height": 1080 },
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/diff.png",
  "originalSize": 4900000,
  "processedSize": 280000
}
```

## Response Fields {#response-fields}

| Field | Type | Description |
|-------|------|-------------|
| jobId | string | ตัวระบุงานสำหรับดาวน์โหลดภาพความแตกต่าง |
| similarity | number | เปอร์เซ็นต์ความคล้ายคลึงระหว่างภาพสองภาพ (0 ถึง 100) |
| dimensions | object | ความกว้างและความสูงที่ใช้ในการเปรียบเทียบ |
| downloadUrl | string | URL สำหรับดาวน์โหลดภาพความแตกต่างที่สร้างขึ้น |
| originalSize | number | ขนาดรวมของภาพอินพุตทั้งสองเป็นไบต์ |
| processedSize | number | ขนาดของภาพเอาต์พุตความแตกต่างเป็นไบต์ |

## Notes {#notes}

- ภาพทั้งสองจะถูกปรับขนาดให้เท่ากัน (ใช้ค่าสูงสุดของแต่ละแกน) ก่อนการเปรียบเทียบ
- ภาพความแตกต่างเน้นความแตกต่างด้วยสีแดง โดยความทึบแสงเป็นสัดส่วนกับขนาดของการเปลี่ยนแปลง พิกเซลที่เหมือนกันหรือเกือบเหมือนกัน (ความแตกต่าง < 10) จะแสดงเป็นภาพต้นฉบับแบบกึ่งโปร่งใส
- ความคล้ายคลึงคำนวณเป็นค่าผกผันของความแตกต่างเฉลี่ยของพิกเซลทั้งหมด แสดงเป็นเปอร์เซ็นต์
- ความคล้ายคลึง 100% หมายความว่าภาพเหมือนกันในระดับพิกเซล (ที่ความละเอียดสำหรับการเปรียบเทียบ)
- เอาต์พุตความแตกต่างเป็นรูปแบบ PNG เสมอไม่ว่ารูปแบบอินพุตจะเป็นอะไร
- ภาพทั้งสองจะถูกตรวจสอบและถอดรหัส (รองรับ HEIC, RAW, PSD, SVG) ก่อนการเปรียบเทียบ
- การจัดวางแนว EXIF จะถูกนำมาใช้อัตโนมัติกับภาพทั้งสองก่อนประมวลผล
