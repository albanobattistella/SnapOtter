# Що збирає SnapOtter {#what-snapotter-collects}

Анонімна аналітика продукту увімкнена за замовчуванням і встановлюється для всього екземпляра адміністратором. Вимкніть її у розділі Settings > System > Privacy.

## Події, які ми надсилаємо (коли увімкнено) {#events-we-send-when-enabled}

- tool_used: id інструмента, статус, тривалість, категорія, чи це AI-інструмент, код помилки у разі збою.
- pipeline_executed: кількість кроків, id інструментів, прапорець пакетної обробки, кількість файлів, тривалість, статус.
- ai_bundle_action: id набору, дія, тривалість.
- Використання фронтенду: які сторінки інструментів відкриваються, додані файли (лише кількість), запущений інструмент, завантаження, збереження, пошук (лише кількість результатів), пакетна обробка.
- Звіти про збої: тип помилки та стек джерела лише з базовими іменами файлів.

## Що ми ніколи не збираємо {#what-we-never-collect}

- Імена чи шляхи файлів
- Вміст файлів
- Текст результату OCR
- Метадані зображень (EXIF)
- Витягнутий текст документів
- Вашу IP-адресу чи ідентичність облікового запису

## Як це вимкнути {#turning-it-off}

Адміністратори: Settings > System > Privacy, вимкніть «Anonymous Product Analytics». Це припиниться негайно для всього екземпляра. Щоб зібрати образ, який ніколи не зможе передавати дані, встановіть аргумент збірки `SNAPOTTER_ANALYTICS=off`.
