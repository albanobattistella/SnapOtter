# Об'єднання CSV {#merge-csvs}

Об'єднання кількох файлів CSV або TSV із однаковими стовпцями в один зведений файл. Усі вхідні файли повинні мати однакові заголовки стовпців.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/files/merge-csvs`

Приймає дані форми у форматі multipart із двома або більше файлами CSV. Поле налаштувань не потрібне.

## Параметри {#parameters}

Цей інструмент не має настроюваних параметрів. Завантажте від 2 до 20 файлів CSV або TSV з однаковими заголовками стовпців.

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/files/merge-csvs \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@january.csv" \
  -F "file=@february.csv" \
  -F "file=@march.csv"
```

## Приклад відповіді {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/merged.csv",
  "originalSize": 30000,
  "processedSize": 28500
}
```

## Примітки {#notes}

- Потрібно від 2 до 20 вхідних файлів.
- Усі файли повинні мати однакові заголовки стовпців. Якщо стовпці не збігаються, об'єднання завершиться помилкою.
- Рядок заголовка включається у вихідний файл один раз; рядки даних з усіх файлів об'єднуються в порядку завантаження.
- Приймаються як файли CSV, так і TSV, але всі файли в одному запиті повинні використовувати однаковий роздільник.
