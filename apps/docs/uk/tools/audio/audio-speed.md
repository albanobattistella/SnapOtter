# Швидкість аудіо {#audio-speed}

Прискорюйте або сповільнюйте відтворення аудіо, застосовуючи множник швидкості.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/audio/audio-speed`

Приймає багаточастинні дані форми з аудіофайлом та JSON-полем `settings`.

## Параметри {#parameters}

| Параметр | Тип | Обов'язковий | Типове значення | Опис |
|-----------|------|----------|---------|-------------|
| factor | number | Ні | `1.5` | Множник швидкості (0.25 до 4). Значення нижче 1 сповільнюють; вище 1 прискорюють. |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/audio/audio-speed \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@audio.mp3" \
  -F 'settings={"factor": 2}'
```

## Приклад відповіді {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/audio.mp3",
  "originalSize": 4500000,
  "processedSize": 2250000
}
```

## Примітки {#notes}

- Множник `0.25` відтворює на чверть швидкості (у 4 рази довше). Множник `4` відтворює на почетверенній швидкості (у 4 рази коротше).
- Висота тону зберігається під час зміни швидкості (розтягнення в часі). Використовуйте зсув висоти тону, щоб коригувати висоту незалежно.
- Вихід зазвичай зберігає вхідний контейнер. Вхід AAC записується як M4A, а непідтримувані входи лише для декодування переходять на MP3.
