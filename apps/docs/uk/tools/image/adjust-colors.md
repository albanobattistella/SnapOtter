# Регулювання кольорів {#adjust-colors}

Комплексний інструмент регулювання кольорів, що поєднує яскравість, контрастність, експозицію, насиченість, температуру, тонування, обертання відтінку, порівневі налаштування окремих каналів та ефекти в один клік (відтінки сірого, сепія, інверсія) в одній кінцевій точці.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/image/adjust-colors`

Приймає дані форми у форматі multipart із файлом зображення та полем JSON `settings`.

## Параметри {#parameters}

| Параметр | Тип | Обов'язковий | Типове значення | Опис |
|-----------|------|----------|---------|-------------|
| brightness | number | Ні | `0` | Регулювання яскравості (-100 до 100) |
| contrast | number | Ні | `0` | Регулювання контрастності (-100 до 100) |
| exposure | number | Ні | `0` | Експозиція / гамма середніх тонів (-100 до 100) |
| saturation | number | Ні | `0` | Насиченість кольору (-100 до 100) |
| temperature | number | Ні | `0` | Баланс білого: холодний/синій до теплого/помаранчевого (-100 до 100) |
| tint | number | Ні | `0` | Зсув тонування: зелений до пурпурового (-100 до 100) |
| hue | number | Ні | `0` | Обертання відтінку у градусах (-180 до 180) |
| sharpness | number | Ні | `0` | Сила різкості (0 до 100) |
| red | number | Ні | `100` | Рівень червоного каналу (0 до 200, 100 = без змін) |
| green | number | Ні | `100` | Рівень зеленого каналу (0 до 200, 100 = без змін) |
| blue | number | Ні | `100` | Рівень синього каналу (0 до 200, 100 = без змін) |
| effect | string | Ні | `"none"` | Колірний ефект: `none`, `grayscale`, `sepia`, `invert` |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/adjust-colors \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"brightness": 20, "contrast": 10, "saturation": -30, "effect": "none"}'
```

Застосувати теплий вінтажний вигляд:

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/adjust-colors \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"temperature": 40, "saturation": -15, "contrast": 10, "effect": "sepia"}'
```

## Примітки {#notes}

- Усі параметри за замовчуванням мають нейтральні значення, тож ви можете регулювати лише те, що потрібно.
- Регулювання застосовуються в такому порядку: яскравість, контрастність, експозиція, насиченість/відтінок, температура/тонування, різкість, канали, ефекти.
- Температура використовує матрицю колірної рекомбінації 3x3 по осях синій-помаранчевий та зелений-пурпуровий.
- Експозиція зіставляється з функцією гамми Sharp (додатні значення освітлюють середні тони, від'ємні затемнюють їх).
- Ця кінцева точка також відповідає на застарілих шляхах `/api/v1/tools/image/brightness-contrast`, `/api/v1/tools/image/saturation`, `/api/v1/tools/image/color-channels` та `/api/v1/tools/image/color-effects`. Усі використовують однакову схему.
- Вихідний формат відповідає вхідному. Вхідні дані HEIC, RAW, PSD та SVG автоматично декодуються перед обробкою.
