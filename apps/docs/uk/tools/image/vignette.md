# Віньєтка {#vignette}

Додайте ефект віньєтки, що затемнює або тонує краї зображення. Підтримує регульовані силу, колір, радіус, м’якість, округлість та положення центру.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/image/vignette`

Приймає багаточастинні дані форми із файлом зображення та полем JSON `settings`.

## Параметри {#parameters}

| Параметр | Тип | Обов’язковий | За замовчуванням | Опис |
|-----------|------|----------|---------|-------------|
| strength | number | Ні | `0.5` | Непрозорість віньєтки (0.1-1) |
| color | string | Ні | `"#000000"` | Шістнадцятковий колір віньєтки |
| radius | integer | Ні | `70` | Зовнішній радіус як відсоток половини діагоналі (0-100) |
| softness | integer | Ні | `50` | М’якість розтушовування (0-100); вищі значення дають більш поступове згасання |
| roundness | integer | Ні | `100` | Форма: 100 = коло, 0 = еліпс відповідно до співвідношення сторін зображення |
| centerX | integer | Ні | `50` | Горизонтальне положення центру у відсотках (0-100) |
| centerY | integer | Ні | `50` | Вертикальне положення центру у відсотках (0-100) |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/vignette \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"strength": 0.7, "radius": 60, "softness": 70}'
```

## Приклад відповіді {#example-response}

```json
{

  "originalSize": 2450000,

```

## Примітки {#notes}

- Менший `radius` затемнює більшу частину зображення; більший радіус обмежує віньєтку крайніми краями.
- Використовуйте нечорний `color` (напр., білий або сепію) для творчих ефектів віньєтки.
- Регулювання `centerX` та `centerY` дозволяє розмістити чисту ділянку зі зміщенням від центру, що корисно для привернення уваги до об’єкта, який не в центрі кадру.
- Формат виводу відповідає формату вводу. Вхідні файли HEIC, RAW, PSD та SVG автоматично декодуються перед обробкою.
