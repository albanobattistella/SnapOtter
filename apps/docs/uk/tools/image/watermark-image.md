# Водяний знак зображенням {#image-watermark}

Накладіть логотип чи вторинне зображення як водяний знак на базове зображення. Водяний знак масштабується відносно ширини базового зображення та розміщується в кутку або центрі.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/image/watermark-image`

Приймає багаточастинні дані форми з **двома** файлами зображень та полем JSON `settings`.

## Параметри {#parameters}

| Параметр | Тип | Обов’язковий | За замовчуванням | Опис |
|-----------|------|----------|---------|-------------|
| position | string | Ні | `"bottom-right"` | Розміщення водяного знака: `center`, `top-left`, `top-right`, `bottom-left`, `bottom-right` |
| opacity | number | Ні | `50` | Відсоток непрозорості водяного знака (від 0 до 100) |
| scale | number | Ні | `25` | Ширина водяного знака як відсоток ширини основного зображення (від 1 до 100) |

### Поля файлів {#file-fields}

| Ім’я поля | Обов’язкове | Опис |
|------------|----------|-------------|
| file | Так | Основне/базове зображення |
| watermark | Так | Зображення водяного знака/логотипа |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/watermark-image \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F "watermark=@logo.png" \
  -F 'settings={"position": "bottom-right", "opacity": 60, "scale": 20}'
```

## Приклад відповіді {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",

  "originalSize": 2450000,
  "processedSize": 2520000
}
```

## Примітки {#notes}

- Обидва зображення перевіряються та декодуються (підтримуються HEIC, RAW, PSD, SVG).
- Водяний знак масштабується пропорційно, тож його ширина дорівнює `scale`% ширини основного зображення.
- Непрозорість застосовується через альфа-маску, скомпоновану зі змішуванням `dest-in`.
- Кутові положення використовують відступ 20px від краю зображення.
- Якщо зображення водяного знака має прозорість (напр., PNG-логотип), вона зберігається під час компонування.
- Орієнтація EXIF автоматично застосовується до обох зображень перед обробкою.
