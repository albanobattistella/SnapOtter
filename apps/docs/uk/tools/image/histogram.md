# Гістограма {#histogram}

Генерує діаграму RGB-гістограми із зображення. Повертає зображення гістограми у форматі PNG разом із поканальною статистикою та сирими даними гістограми на 256 бінів у JSON-відповіді.

## API Endpoint {#api-endpoint}

`POST /api/v1/tools/image/histogram`

Приймає дані форми multipart з файлом зображення та полем JSON `settings`.

## Параметри {#parameters}

| Параметр | Тип | Обовʼязковий | За замовчуванням | Опис |
|-----------|------|----------|---------|-------------|
| scale | string | Ні | `"linear"` | Масштаб осі Y: `linear` або `log` |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/histogram \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"scale": "linear"}'
```

## Примітки {#notes}

- `downloadUrl` вказує на відрендерену діаграму гістограми PNG, що показує розподіли R, G, B та яскравості.
- `bins` містить сирі масиви з 256 значень для кожного каналу (червоний, зелений, синій, яскравість), придатні для рендерингу власних візуалізацій.
- `stats` надає середнє, медіану та стандартне відхилення для кожного каналу.
- `mean` та `max` — це поля скороченого запису для зворотної сумісності.
- Використовуйте масштаб `log`, коли в гістограмі домінує кілька піків, а ви хочете побачити деталі в нижчих бінах.
- Вхідні дані HEIC, RAW, PSD та SVG автоматично декодуються перед аналізом.
