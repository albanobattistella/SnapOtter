# Рамка та обрамлення {#border-frame}

Додавання рамок, відступів, заокруглених кутів та падаючих тіней до зображень. Інструмент застосовує ефекти в порядку: відступ, рамка, радіус кутів, потім тінь.

## Кінцева точка API {#api-endpoint}

`POST /api/v1/tools/image/border`

## Параметри {#parameters}

| Параметр | Тип | Обов'язковий | Типове значення | Опис |
|-----------|------|----------|---------|-------------|
| borderWidth | number | Ні | 10 | Товщина рамки в пікселях (0 до 2000) |
| borderColor | string | Ні | `"#000000"` | Колір рамки як hex (наприклад, `#FF0000`) |
| padding | number | Ні | 0 | Внутрішній відступ між зображенням та рамкою в пікселях (0 до 200) |
| paddingColor | string | Ні | `"#FFFFFF"` | Колір заповнення відступу як hex |
| cornerRadius | number | Ні | 0 | Радіус кутів у пікселях (0 до 2000) |
| shadow | boolean | Ні | `false` | Чи додавати падаючу тінь |
| shadowBlur | number | Ні | 15 | Радіус розмиття тіні (1 до 200) |
| shadowOffsetX | number | Ні | 0 | Горизонтальний зсув тіні (-50 до 50) |
| shadowOffsetY | number | Ні | 5 | Вертикальний зсув тіні (-50 до 50) |
| shadowColor | string | Ні | `"#000000"` | Колір тіні як hex |
| shadowOpacity | number | Ні | 40 | Відсоток непрозорості тіні (0 до 100) |

## Приклад запиту {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/border \
  -F "file=@photo.jpg" \
  -F 'settings={"borderWidth":20,"borderColor":"#333333","cornerRadius":16,"shadow":true,"shadowBlur":25,"shadowOpacity":50}'
```

## Примітки {#notes}

- Використовує стандартну фабрику `createToolRoute`. Приймає один файл зображення через завантаження multipart.
- Підтримує вхідні формати HEIC, RAW, PSD та SVG (автоматично декодуються).
- Порядок обробки: спочатку додається відступ, потім рамка обгортає навколо, потім застосовується радіус кутів, потім компонується тінь.
- Коли `cornerRadius` або `shadow` увімкнено, вивід примусово встановлюється у PNG (незалежно від вхідного формату) для збереження прозорості. Формати, що підтримують альфа-канал (PNG, WebP, AVIF), зберігають свій початковий формат.
- Тінь враховує форму: вона слідує за заокругленими кутами, а не створює прямокутну тінь.
- Встановлення `borderWidth` на 0 та використання лише `cornerRadius` + `shadow` створює безрамковий заокруглений ефект тіні.
