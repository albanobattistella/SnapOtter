# Что собирает SnapOtter {#what-snapotter-collects}

Анонимная продуктовая аналитика включена по умолчанию и задаётся для всего экземпляра администратором. Отключите её в разделе Settings > System > Privacy.

## События, которые мы отправляем (когда включено) {#events-we-send-when-enabled}

- tool_used: id инструмента, статус, длительность, категория, является ли он инструментом ИИ, код ошибки при сбое.
- pipeline_executed: количество шагов, id инструментов, флаг пакетной обработки, количество файлов, длительность, статус.
- ai_bundle_action: id набора, действие, длительность.
- Использование интерфейса: какие страницы инструментов открываются, добавленные файлы (только количество), запуск инструмента, загрузки, сохранения, поиск (только количество результатов), пакетная обработка.
- Отчёты о сбоях: тип ошибки и стек исходников только с базовыми именами файлов.

## Что мы никогда не собираем {#what-we-never-collect}

- Имена или пути файлов
- Содержимое файлов
- Текст вывода OCR
- Метаданные изображений (EXIF)
- Извлечённый текст документов
- Ваш IP-адрес или идентичность учётной записи

## Как отключить {#turning-it-off}

Администраторам: Settings > System > Privacy, переключите «Anonymous Product Analytics» в положение «выключено». Сбор прекращается немедленно, для всего экземпляра. Чтобы собрать образ, который никогда не сможет ничего отправлять, задайте аргумент сборки `SNAPOTTER_ANALYTICS=off`.
