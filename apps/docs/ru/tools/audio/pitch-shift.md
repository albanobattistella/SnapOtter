# Сдвиг высоты тона {#pitch-shift}

Повышайте или понижайте высоту тона аудиофайла на определённое число полутонов без изменения скорости воспроизведения.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/audio/pitch-shift`

Принимает данные формы multipart с аудиофайлом и полем `settings` в формате JSON.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| semitones | integer | Нет | `3` | Полутоны для сдвига (от -12 до 12). Должно быть отлично от нуля. |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/audio/pitch-shift \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@audio.mp3" \
  -F 'settings={"semitones": -5}'
```

## Пример ответа {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/audio.mp3",
  "originalSize": 4500000,
  "processedSize": 4500000
}
```

## Примечания {#notes}

- Положительные значения повышают высоту тона; отрицательные понижают её.
- Сдвиг на 12 полутонов равен одной октаве вверх; -12 равен одной октаве вниз.
- Длительность воспроизведения остаётся неизменной независимо от величины сдвига.
- Вывод обычно сохраняет входной контейнер. Вход AAC записывается как M4A, а неподдерживаемые для декодирования входы откатываются к MP3.
