# Повышение резкости {#sharpening}

Продвинутый инструмент повышения резкости с тремя методами: адаптивный (умный, с учётом границ), нерезкое маскирование (классические радиус/степень) и высокочастотный фильтр (усиление текстуры). Включает встроенное шумоподавление для предотвращения артефактов резкости.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/sharpening`

Принимает multipart form data с файлом изображения и JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| method | string | Нет | `"adaptive"` | Алгоритм повышения резкости: `adaptive`, `unsharp-mask`, `high-pass` |
| sigma | number | Нет | `1.0` | Адаптивный: сигма Гаусса (от 0.5 до 10) |
| m1 | number | Нет | `1.0` | Адаптивный: повышение резкости в ровных областях (от 0 до 10) |
| m2 | number | Нет | `3.0` | Адаптивный: повышение резкости в зубчатых областях (от 0 до 20) |
| x1 | number | Нет | `2.0` | Адаптивный: порог ровных/зубчатых областей (от 0 до 10) |
| y2 | number | Нет | `12` | Адаптивный: максимальное повышение резкости в ровных областях (от 0 до 50) |
| y3 | number | Нет | `20` | Адаптивный: максимальное повышение резкости в зубчатых областях (от 0 до 50) |
| amount | number | Нет | `100` | Нерезкое маскирование: степень повышения резкости (от 0 до 1000) |
| radius | number | Нет | `1.0` | Нерезкое маскирование: радиус размытия в пикселях (от 0.1 до 5) |
| threshold | number | Нет | `0` | Нерезкое маскирование: минимальная разница яркости для повышения резкости (от 0 до 255) |
| strength | number | Нет | `50` | Высокочастотный: сила фильтра (от 0 до 100) |
| kernelSize | number | Нет | `3` | Высокочастотный: размер ядра свёртки (3 или 5) |
| denoise | string | Нет | `"off"` | Шумоподавление перед повышением резкости: `off`, `light`, `medium`, `strong` |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/sharpening \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"method": "adaptive", "sigma": 1.5}'
```

Нерезкое маскирование с порогом для защиты плавных областей:

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/sharpening \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"method": "unsharp-mask", "amount": 150, "radius": 1.5, "threshold": 10}'
```

## Примечания {#notes}

- Используются только те параметры, которые относятся к выбранному методу. Например, `amount`, `radius` и `threshold` игнорируются, когда `method` равно `adaptive`.
- Адаптивный метод использует встроенное адаптивное повышение резкости Sharp с настраиваемым поведением для ровных/зубчатых областей.
- Опция `denoise` применяет шумоподавление перед повышением резкости, чтобы предотвратить усиление шума/зерна.
- Высокочастотное повышение резкости извлекает мелкие детали, вычитая размытую версию из оригинала, а затем смешивая обратно.
- Формат вывода соответствует формату ввода. Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед обработкой.
