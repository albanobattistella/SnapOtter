# Водяной знак-изображение {#image-watermark}

Наложите логотип или дополнительное изображение в качестве водяного знака на базовое изображение. Водяной знак масштабируется относительно ширины базового изображения и размещается в углу или по центру.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/watermark-image`

Принимает multipart form data с **двумя** файлами изображений и JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| position | string | Нет | `"bottom-right"` | Размещение водяного знака: `center`, `top-left`, `top-right`, `bottom-left`, `bottom-right` |
| opacity | number | Нет | `50` | Процент непрозрачности водяного знака (от 0 до 100) |
| scale | number | Нет | `25` | Ширина водяного знака как процент от ширины основного изображения (от 1 до 100) |

### Поля файлов {#file-fields}

| Имя поля | Обязательное | Описание |
|------------|----------|-------------|
| file | Да | Основное/базовое изображение |
| watermark | Да | Изображение водяного знака/логотипа |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/watermark-image \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F "watermark=@logo.png" \
  -F 'settings={"position": "bottom-right", "opacity": 60, "scale": 20}'
```

## Пример ответа {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/photo.jpg",
  "originalSize": 2450000,
  "processedSize": 2520000
}
```

## Примечания {#notes}

- Оба изображения проверяются и декодируются (поддерживаются HEIC, RAW, PSD, SVG).
- Водяной знак масштабируется пропорционально так, что его ширина равна `scale`% от ширины основного изображения.
- Непрозрачность применяется через альфа-маску, композитную со смешиванием `dest-in`.
- Угловые позиции используют отступ 20px от края изображения.
- Если изображение водяного знака имеет прозрачность (например, PNG-логотип), она сохраняется при композитинге.
- Ориентация EXIF автоматически применяется к обоим изображениям перед обработкой.
