# Виньетка {#vignette}

Добавьте эффект виньетки, который затемняет или тонирует края изображения. Поддерживает регулируемые силу, цвет, радиус, мягкость, скругление и положение центра.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/vignette`

Принимает multipart form data с файлом изображения и JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| strength | number | Нет | `0.5` | Непрозрачность виньетки (0.1-1) |
| color | string | Нет | `"#000000"` | Цвет виньетки в шестнадцатеричном формате |
| radius | integer | Нет | `70` | Внешний радиус как процент от половины диагонали (0-100) |
| softness | integer | Нет | `50` | Мягкость растушёвки (0-100); более высокие значения дают более плавное затухание |
| roundness | integer | Нет | `100` | Форма: 100 = круг, 0 = эллипс, соответствующий соотношению сторон изображения |
| centerX | integer | Нет | `50` | Горизонтальное положение центра в процентах (0-100) |
| centerY | integer | Нет | `50` | Вертикальное положение центра в процентах (0-100) |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/vignette \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"strength": 0.7, "radius": 60, "softness": 70}'
```

## Пример ответа {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/photo.jpg",
  "originalSize": 2450000,
  "processedSize": 2410000
}
```

## Примечания {#notes}

- Меньший `radius` затемняет большую часть изображения; больший радиус ограничивает виньетку крайними краями.
- Используйте не чёрный `color` (например, белые или сепийные тона) для творческих эффектов виньетки.
- Регулировка `centerX` и `centerY` позволяет смещать светлую область от центра, что полезно для привлечения внимания к объекту, который не находится в середине кадра.
- Формат вывода соответствует формату ввода. Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед обработкой.
