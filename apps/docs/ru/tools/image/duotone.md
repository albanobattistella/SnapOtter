# Дуотон {#duotone}

Применяйте двухцветный эффект дуотон к изображению. Изображение преобразуется в оттенки серого, затем сопоставляется с градиентом между цветом теней (тёмные тона) и цветом светов (яркие тона).

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/duotone`

Принимает multipart-данные формы с файлом изображения и JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| shadow | string | Нет | `"#1e3a8a"` | Hex-цвет теней (применяется к тёмным тонам) |
| highlight | string | Нет | `"#fbbf24"` | Hex-цвет светов (применяется к ярким тонам) |
| intensity | integer | Нет | `100` | Интенсивность эффекта (0-100); 0 возвращает оригинал, 100 применяет полный дуотон |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/duotone \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"shadow": "#0f172a", "highlight": "#f97316", "intensity": 80}'
```

## Пример ответа {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "downloadUrl": "/api/v1/download/a1b2c3d4-e5f6-7890-abcd-ef1234567890/photo.jpg",
  "originalSize": 2450000,
  "processedSize": 1870000
}
```

## Примечания {#notes}

- Формат вывода соответствует формату ввода. Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед обработкой.
- Значение `intensity` менее 100 смешивает результат дуотона с исходным изображением, позволяя получить более тонкие эффекты.
- Популярные комбинации дуотона включают тёмно-синий/золотой, бирюзовый/коралловый и фиолетовый/розовый.
