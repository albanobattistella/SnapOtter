# Регулировка цвета {#adjust-colors}

Комплексный инструмент регулировки цвета, объединяющий яркость, контрастность, экспозицию, насыщенность, температуру, тон, поворот оттенка, поканальные уровни и эффекты в один клик (оттенки серого, сепия, инверсия) в одной конечной точке.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/adjust-colors`

Принимает multipart form data с файлом изображения и полем JSON `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| brightness | number | Нет | `0` | Регулировка яркости (от -100 до 100) |
| contrast | number | Нет | `0` | Регулировка контрастности (от -100 до 100) |
| exposure | number | Нет | `0` | Экспозиция / гамма средних тонов (от -100 до 100) |
| saturation | number | Нет | `0` | Насыщенность цвета (от -100 до 100) |
| temperature | number | Нет | `0` | Баланс белого: холодный/синий к тёплому/оранжевому (от -100 до 100) |
| tint | number | Нет | `0` | Сдвиг тона: от зелёного к пурпурному (от -100 до 100) |
| hue | number | Нет | `0` | Поворот оттенка в градусах (от -180 до 180) |
| sharpness | number | Нет | `0` | Сила резкости (от 0 до 100) |
| red | number | Нет | `100` | Уровень красного канала (от 0 до 200, 100 = без изменений) |
| green | number | Нет | `100` | Уровень зелёного канала (от 0 до 200, 100 = без изменений) |
| blue | number | Нет | `100` | Уровень синего канала (от 0 до 200, 100 = без изменений) |
| effect | string | Нет | `"none"` | Цветовой эффект: `none`, `grayscale`, `sepia`, `invert` |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/adjust-colors \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"brightness": 20, "contrast": 10, "saturation": -30, "effect": "none"}'
```

Применить тёплый винтажный вид:

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/adjust-colors \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"temperature": 40, "saturation": -15, "contrast": 10, "effect": "sepia"}'
```

## Примечания {#notes}

- Все параметры по умолчанию имеют нейтральные значения, поэтому вы можете регулировать только то, что нужно.
- Регулировки применяются в следующем порядке: яркость, контрастность, экспозиция, насыщенность/оттенок, температура/тон, резкость, каналы, эффекты.
- Температура использует матрицу цветовой рекомбинации 3x3 по осям синий-оранжевый и зелёный-пурпурный.
- Экспозиция сопоставляется с гамма-функцией Sharp (положительные значения осветляют средние тона, отрицательные затемняют их).
- Эта конечная точка также отвечает на устаревших путях `/api/v1/tools/image/brightness-contrast`, `/api/v1/tools/image/saturation`, `/api/v1/tools/image/color-channels` и `/api/v1/tools/image/color-effects`. Все они используют одну и ту же схему.
- Выходной формат совпадает с форматом ввода. Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед обработкой.
