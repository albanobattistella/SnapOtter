# Цветовая палитра {#color-palette}

Извлекайте доминирующие цвета из изображения и получайте их в виде hex-значений цвета. Использует квантованный частотный анализ для определения наиболее заметных и визуально различимых цветов.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/color-palette`

Принимает multipart-данные формы с файлом изображения и необязательным JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| count | integer | Нет | `8` | Количество извлекаемых цветов (2-16) |
| format | string | Нет | `"hex"` | Формат цвета: `hex`, `rgb`, `hsl` |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/color-palette \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"count": 6, "format": "hex"}'
```

## Пример ответа {#example-response}

```json
{
  "filename": "photo.jpg",
  "colors": [
    "#304080",
    "#e0a060",
    "#f0f0f0",
    "#203020",
    "#a0c0e0",
    "#806040"
  ],
  "hex": [
    "#304080",
    "#e0a060",
    "#f0f0f0",
    "#203020",
    "#a0c0e0",
    "#806040"
  ],
  "count": 6
}
```

## Поля ответа {#response-fields}

| Поле | Тип | Описание |
|-------|------|-------------|
| filename | string | Очищенное имя файла |
| colors | array | Массив строк цвета в запрошенном формате, упорядоченный по доминированию (сначала наиболее частые) |
| hex | array | Массив hex-строк цвета (всегда hex, независимо от настройки `format`) |
| count | number | Количество извлечённых цветов |

## Примечания {#notes}

- Возвращает до `count` доминирующих цветов (по умолчанию 8, диапазон 2-16), отсортированных по частоте (сначала наиболее частые).
- Изображение внутренне уменьшается до 100x100 пикселей для анализа, поэтому палитра отражает общее распределение цветов, а не мелкие детали.
- Цвета извлекаются с помощью квантования методом медианного сечения, которое рекурсивно разбивает совокупности пикселей вдоль канала с наибольшим диапазоном.
- Альфа-канал удаляется перед анализом, поэтому прозрачные области не учитываются.
- Это конечная точка только для чтения. Она не создаёт загружаемый выходной файл или `jobId`.
- Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед анализом.
