# Имитация дальтонизма {#color-blindness-simulation}

Имитируйте нарушение цветового зрения (CVD), чтобы увидеть, как изображения выглядят для людей с различными типами дальтонизма. Полезно для тестирования доступности дизайнов, диаграмм и интерфейсов.

## Конечная точка API {#api-endpoint}

`POST /api/v1/tools/image/color-blindness`

Принимает multipart-данные формы с файлом изображения и JSON-полем `settings`.

## Параметры {#parameters}

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|-----------|------|----------|---------|-------------|
| simulationType | string | Нет | `"deuteranomaly"` | Тип имитируемого нарушения цветового зрения |

### Типы имитации {#simulation-types}

| Значение | Состояние | Описание |
|-------|-----------|-------------|
| `protanopia` | Отсутствие восприятия красного | Полное отсутствие красных колбочек |
| `deuteranopia` | Отсутствие восприятия зелёного | Полное отсутствие зелёных колбочек |
| `tritanopia` | Отсутствие восприятия синего | Полное отсутствие синих колбочек |
| `protanomaly` | Ослабленное восприятие красного | Сниженная чувствительность красных колбочек |
| `deuteranomaly` | Ослабленное восприятие зелёного | Сниженная чувствительность зелёных колбочек (наиболее распространённое) |
| `tritanomaly` | Ослабленное восприятие синего | Сниженная чувствительность синих колбочек |
| `achromatopsia` | Полная цветовая слепота | Полное отсутствие цветового зрения |
| `blueConeMonochromacy` | Только синие колбочки | Функционируют только синие колбочки |

## Пример запроса {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/color-blindness \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@design.png" \
  -F 'settings={"simulationType": "deuteranopia"}'
```

## Пример ответа {#example-response}

## Примечания {#notes}

- Дейтераномалия (ослабленное восприятие зелёного) используется по умолчанию, поскольку это наиболее распространённая форма нарушения цветового зрения, затрагивающая примерно 6% мужчин.
- Имитация использует матрицы цветового преобразования, моделирующие, как сниженные или отсутствующие колбочковые фоторецепторы изменяют воспринимаемые цвета.
- Этот инструмент неразрушающий и создаёт только предпросмотр. Он не изменяет исходное изображение для доступности.
- Формат вывода соответствует формату ввода. Входные данные HEIC, RAW, PSD и SVG автоматически декодируются перед обработкой.
