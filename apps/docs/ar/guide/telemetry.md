# ما يجمعه SnapOtter {#what-snapotter-collects}

تحليلات المنتج المجهولة مفعّلة افتراضياً ويضبطها المسؤول للمثيل بأكمله. أوقفها تحت Settings > System > Privacy.

## الأحداث التي نرسلها (عند التفعيل) {#events-we-send-when-enabled}

- tool_used: معرّف الأداة والحالة والمدة والفئة، وما إذا كانت أداة ذكاء اصطناعي، ورمز خطأ عند الفشل.
- pipeline_executed: عدد الخطوات ومعرّفات الأدوات وراية الدفعة وعدد الملفات والمدة والحالة.
- ai_bundle_action: معرّف الحزمة والإجراء والمدة.
- استخدام الواجهة الأمامية: أي صفحات أدوات تُفتَح، والملفات المُضافة (أعداد فقط)، وبدء الأداة، والتنزيلات، وعمليات الحفظ، والبحث (عدد النتائج فقط)، والدفعات المعالَجة.
- تقارير الأعطال: نوع الخطأ ومكدّس مصدر يحتوي على أسماء الملفات الأساسية فقط.

## ما لا نجمعه أبداً {#what-we-never-collect}

- أسماء الملفات أو مساراتها
- محتويات الملفات
- نص مخرجات OCR
- بيانات الصور الوصفية (EXIF)
- نص المستندات المستخرَج
- عنوان IP الخاص بك أو هوية حسابك

## إيقافها {#turning-it-off}

المسؤولون: Settings > System > Privacy، أوقِف "Anonymous Product Analytics". تتوقّف فوراً على مستوى المثيل بأكمله. لبناء صورة لا يمكنها الإرسال أبداً، اضبط وسيط البناء `SNAPOTTER_ANALYTICS=off`.
