# التظليل الحوافي {#vignette}

أضف تأثير تظليل حوافي يعتّم حواف الصورة أو يلوّنها. يدعم قوة ولونًا ونصف قطر ونعومة واستدارة وموضع مركز قابلة للتعديل.

## نقطة نهاية API {#api-endpoint}

`POST /api/v1/tools/image/vignette`

تقبل بيانات نموذج متعدد الأجزاء تحتوي على ملف صورة وحقل `settings` بصيغة JSON.

## المعاملات {#parameters}

| المعامل | النوع | مطلوب | الافتراضي | الوصف |
|-----------|------|----------|---------|-------------|
| strength | number | لا | `0.5` | عتامة التظليل الحوافي (0.1-1) |
| color | string | لا | `"#000000"` | لون التظليل بصيغة hex |
| radius | integer | لا | `70` | نصف القطر الخارجي كنسبة مئوية من نصف القطر القُطري (0-100) |
| softness | integer | لا | `50` | نعومة التدرّج (0-100)؛ القيم الأعلى تنتج تلاشيًا أكثر تدرّجًا |
| roundness | integer | لا | `100` | الشكل: 100 = دائرة، 0 = قطع ناقص يطابق نسبة أبعاد الصورة |
| centerX | integer | لا | `50` | موضع المركز الأفقي كنسبة مئوية (0-100) |
| centerY | integer | لا | `50` | موضع المركز العمودي كنسبة مئوية (0-100) |

## مثال طلب {#example-request}

```bash
curl -X POST http://localhost:1349/api/v1/tools/image/vignette \
  -H "Authorization: Bearer si_your-api-key" \
  -F "file=@photo.jpg" \
  -F 'settings={"strength": 0.7, "radius": 60, "softness": 70}'
```

## مثال استجابة {#example-response}

```json
{
  "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",

  "originalSize": 2450000,

```

## ملاحظات {#notes}

- قيمة `radius` الأصغر تعتّم جزءًا أكبر من الصورة؛ والقيمة الأكبر تحصر التظليل في الحواف القصوى.
- استخدم `color` غير أسود (مثل درجات البيضاء أو السيبيا) للحصول على تأثيرات تظليل إبداعية.
- يتيح لك تعديل `centerX` و`centerY` وضع المنطقة الصافية خارج المركز، وهو مفيد لتوجيه التركيز نحو موضوع لا يقع في منتصف الإطار.
- تطابق صيغة المخرجات صيغة المدخلات. تُفكّ ترميزات مدخلات HEIC وRAW وPSD وSVG تلقائيًا قبل المعالجة.
