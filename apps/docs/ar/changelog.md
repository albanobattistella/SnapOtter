# سجل التغييرات {#changelog}

## v2.0.0 {#v2-0-0}

يحوّل الإصدار SnapOtter 2.0 مجموعة أدوات الصور إلى مجموعة كاملة لمعالجة الملفات: أكثر من 200 أداة عبر خمس وسائط (الصور والفيديو والصوت وPDF والملفات)، أُعيد بناؤها على Postgres 17 وطابور مهام مدعوم بـ Redis، مع `docker run` بأمر واحد. هذا إصدار رئيسي؛ اقرأ التغييرات الجذرية قبل الترقية من الإصدار 1.x.

### ميزات جديدة {#new-features}

- **أربع وسائط أدوات جديدة**: تنضم الفيديو والصوت وPDF والملفات إلى الصور، لترفع الفهرس إلى أكثر من 200 أداة.
- **مهام خلفية دائمة**: طابور مدعوم بـ Redis (BullMQ) يشغّل كل أداة كمهمة متتبَّعة مع تقدّم SSE مباشر.
- **وضع الحاوية الواحدة المتكامل**: أمر `docker run` واحد يقلع نسخة كاملة مع Postgres وRedis مدمجين.
- **حزم ذكاء اصطناعي عند الطلب**: إزالة الخلفية، والتعرّف الضوئي على الحروف OCR، والنسخ النصي، والتكبير، واكتشاف الوجوه وتحسينها، وممحاة الكائنات، والتلوين، وترميم الصور تُثبَّت من الواجهة. يُكتشف تسريع GPU لكل إطار عمل على حدة.
- **توقيع PDF**: ارسم توقيعك أو اكتبه أو ارفعه وضعه على ملف PDF داخل المتصفح.
- **الأتمتة**: أداة بصرية لبناء خطوط الأنابيب تربط الأدوات معًا، مع تسع قوالب جاهزة.
- **83 إعداد تحويل مسبق بنقرة واحدة**: محوّلات مخصّصة من JPG إلى PNG، ومن MP4 إلى GIF، وما شابه، مع بحث تقريبي.
- **محرر صور قائم على الطبقات**: محرر مدعوم بـ Konva في `/editor` مع فُرَش وأشكال وتعديلات ومرشحات ومنحنيات.
- **مكتبة الملفات**: احفظ أي نتيجة وأعد استخدامها كمُدخَل لأداة أخرى.
- الأدوات المثبَّتة، والتكبير والتحريك داخل اللوحة، و21 لغة، وقدرات المؤسسات (OIDC/SSO وSAML وSCIM وتخزين S3 وأذونات لكل أداة وتصدير سجل التدقيق والتتبّع الموزَّع).

### تحسينات {#improvements}

- إلغاء عملية قيد التشغيل. (#137)
- فك ترميز RAW بدقة كاملة عبر LibRaw، بما في ذلك DNG. (#289)
- عمليات النشر غير الجذرية وبمعرّف مستخدم أجنبي (TrueNAS وUnraid وOpenShift وPUID/PGID). (#230، #127)
- اكتشاف دقيق لتثبيت الذكاء الاصطناعي وتدفّق تثبيت مُحصَّن. (#214، #352)
- تعزيز الخصوصية: لا خروج تلقائي إلى أطراف ثالثة، إضافة إلى وضع عدم اتصال صارم اختياري.
- زر تعليقات دائم التفعيل، حتى مع إيقاف التحليلات.

### إصلاحات الأخطاء {#bug-fixes}

- `RATE_LIMIT_PER_MIN=0` يعطّل تحديد المعدل لمسارات الأدوات مجددًا. (#271)
- إصلاح مسارات البيئة الافتراضية للذكاء الاصطناعي داخل صورة Docker. (#390)
- التوافق مع sharp 0.35.2 فما فوق. (#362)
- إصلاحات تخطيط محرر الصور: المساطر وسلوك التعبئة والشريط الجانبي وتحجيم اللوحة. (#258، #259)
- استكمال الترجمة الإيطالية. (#231، #206، #425)
- تطبيع الصوت وloudnorm يحافظان على معدّل العينات المصدري.
- تعزيز ضد SSRF: مطابقة CIDR الرقمية لـ IPv6 ومسح مسبق موسَّع لعناوين URL. (#287)
- ملفات PDF المُولَّدة تُختَم بـ SnapOtter كمُنتِج (Producer).
- تثبيت mediapipe على Python 3.13 وDebian 13.

### تغييرات جذرية {#breaking-changes}

يستبدل الإصدار 2.0 قاعدة بيانات SQLite المدمجة بـ Postgres 17 ويضيف Redis 8 لطابور المهام. تُرحَّل بيانات الإصدار 1.x تلقائيًا عند الإقلاع الأول، لكن حزمة الحاويات تغيّرت، لذا انسخ احتياطيًا وحدة تخزين `/data` بأكملها أولًا (يشغّل الإصدار 1.x قاعدة SQLite في وضع WAL، لذا تكون البيانات المُثبَّتة عادةً في `snapotter.db-wal`). ثم اختر صورة الحاوية الواحدة (Postgres وRedis مدمجين، للجذر فقط) أو حزمة Compose (التطبيق إضافة إلى Postgres 17 وRedis 8). راجع [دليل الترحيل](https://github.com/snapotter-hq/SnapOtter/blob/main/MIGRATING.md) و[دليل الترقية](/ar/guide/upgrading).

## v1.17.2 {#v1-17-2}

أداة جديدة لتحويل HTML إلى صورة، وإتاحة وصول WCAG 2.2 AA، وتعزيز أمني من اختبار الاختراق، و5 إصلاحات حرجة لـ Docker.

### ميزات جديدة {#new-features-1}

- **HTML إلى صورة**: التقط لقطات شاشة لعناوين URL أو لـ HTML خام بصيغة PNG/JPEG/WebP. التقاطات للصفحة الكاملة، ومنافذ عرض مخصّصة، ووضع داكن.
- **اصطلاح Docker للأسرار _FILE**: ثبّت متغيّرات البيئة الحساسة كملفات بدل النص الصريح. (#205)
- **ترخيص المؤسسات وتخزين S3**: مفتاح ترخيص تجاري اختياري وتخزين كائنات متوافق مع S3.
- **تحسينات محرر الأشكال**: شفافية التعبئة/الحدّ، ومنتقي ألوان RGBA، وأنماط الخطوط المتقطّعة.
- **أرشيفات إصدار جاهزة**: نزّل حزم tarball من GitHub Releases للتثبيتات غير المعتمدة على Docker (Proxmox، الأجهزة الفعلية، LXC). (#202)

### تحسينات {#improvements-1}

- **إتاحة وصول WCAG 2.2 AA**: تخطّي التنقّل، وحبس التركيز، ومناطق aria-live، ودعم تقليل الحركة، ونسب تباين صحيحة. (#209)
- **الاستجابة على الأجهزة المحمولة**: إعدادات مستجيبة، وإعادة اتصال SSE تلقائيًا عند تبديل تبويب الجوّال. (#203، #204)
- **جودة إزالة الخلفية**: تنعيم الحواف، وإزالة تلوّث الألوان، واختيار صيغة الإخراج.
- **الترجمة الإيطالية**: نحو 145 سلسلة جديدة بواسطة @albanobattistella. (#206)
- **توثيق API لكل أداة**: 53 صفحة توثيق مع المعاملات والأمثلة وصيغ الاستجابة.
- **تنزيلات نماذج الذكاء الاصطناعي**: منطق إعادة المحاولة مع تراجع أسّي لـ HuggingFace. (#201)

### إصلاحات الأخطاء {#bug-fixes-1}

- كانت حاويات Docker الجديدة غير قابلة للاستخدام تمامًا (حجب تحديد المعدل كل الطلبات).
- فشلت أدوات اكتشاف الوجوه بالذكاء الاصطناعي (blur-faces وred-eye-removal وenhance-faces وpassport-photo) على جميع المنصّات.
- تعطّلت ملفات HEIC على ARM (عدم تطابق رمز libheif).
- فشل تثبيت حزم التكبير وترميم الصور بالذكاء الاصطناعي على ARM.
- استخدم OCR إصدار CUDA خاطئًا على حاويات GPU.
- تجاوز حارس SSRF عبر عناوين IPv6 المُطابِقة لـ IPv4 بصيغة سداسية عشرية. (بفضل: @tonghuaroot)
- فك ترميز HEIC من iPhone مع صور مساعِدة. (#183، #199)
- نفاد ذاكرة CUDA في Real-ESRGAN على بطاقات GPU سعة 8GB. (#200)
- 6 أخطاء إنتاجية في Sentry و7 أخطاء ضمان جودة. (#208)

### الأمان {#security}

- معالجة 10 نتائج اختبار اختراق (تجاوز XFF، وانهيارات JSON المُشوَّه، وخطوط الأنابيب غير المحدودة، وXSS في سجل التدقيق، وطريقة TRACE، وغير ذلك). (#207)
- حجب تجاوز SSRF عبر IPv6 السداسي العشري. (بفضل: @tonghuaroot)
- تثبيت صور Dockerfile الأساسية بحسب البصمة الرقمية (digest).

## v1.17.0 {#v1-17-0}

خمس أدوات جديدة، ومحرر صور كامل، وتسجيل دخول SSO، و20 لغة. ربما كان يجب أن تكون ثلاثة إصدارات منفصلة، لكن ها نحن أولاء.

### ميزات جديدة {#new-features-3}

- **محرر الصور** - طبقات وفُرَش وأشكال وتعديلات ومرشحات ومنحنيات واختصارات لوحة مفاتيح. يعمل في متصفحك، ويعالج على عتادك.
- **مصادقة OIDC / SSO** - سجّل الدخول عبر Google أو GitHub أو Okta أو أي مزوّد OpenID Connect. اضبط بضعة متغيّرات بيئة ويستخدم فريقك حساباتهم القائمة.
- **مولّد الميمات** - 100 قالب مدمج مع عرض النص عبر opentype.js. أو ارفع صورتك الخاصة.
- **التجميل** - أفلِت لقطة شاشة، واحصل على صورة مصقولة. إطارات أجهزة (macOS وWindows والمتصفح)، وظلال، وتدرّجات، وإعدادات جاهزة لوسائل التواصل الاجتماعي.
- **محاكاة عمى الألوان** - عايِن كيف تبدو الصور مع عمى الأحمر (protanopia) وعمى الأخضر (deuteranopia) وعمى الأزرق (tritanopia) وغيرها من قصور رؤية الألوان.
- **مُصلِح شفافية PNG** - يكتشف ملفات PNG الشفافة زيفًا ويصلحها بمَطّ BiRefNet عالي الدقة. إزالة العلامة المائية اختياريًا عبر ملء LaMa.
- **توسيع لوحة الرسم بالذكاء الاصطناعي** - وسّع حدود الصورة بملء ذكي. ثلاث فئات جودة (سريع، متوازن، عالي الجودة) بحسب مقدار وقت GPU الذي تريد المقايضة به.
- **20 لغة** - العربية، والصينية (المبسّطة/التقليدية)، والتشيكية، والهولندية، والفرنسية، والألمانية، والهندية، والإندونيسية، والإيطالية، واليابانية، والكورية، والبولندية، والبرتغالية، والروسية، والإسبانية، والتايلاندية، والتركية، والأوكرانية، والفيتنامية. يعمل الاتجاه من اليمين إلى اليسار للعربية.
- **الاستيراد من عنوان URL** - الصق عناوين URL في منطقة الإفلات أو استوردها دفعةً من قائمة. جلب من جانب الخادم مع حماية ضد SSRF.
- **ممحاة متعددة الملفات** - ارسم أقنعة المسح عبر صور متعددة، وعالجها جميعًا بنقرة واحدة. تبقى الضربات محفوظة لكل صورة.
- **استيراد/تصدير خط الأنابيب** - احفظ سلاسل الأدوات كـ JSON، وشاركها مع الآخرين.
- **17 صيغة RAW جديدة للكاميرات** عبر exiftool، إضافة إلى إدخال QOI وJP2 وEPS وDDS وCUR وDPX وFITS وPPM/PGM/PBM وSVGZ وAPNG. مُرمِّزات إخراج جديدة لـ BMP وICO وJP2 وQOI. واستُعيد تصدير AVIF وTIFF وGIF وJXL وPSD من فرع مفقود سابقًا.

### تحسينات {#improvements-2}

- **تحسين الصورة** - استُبدل خط الأنابيب القديم بـ CLAHE + التطبيع + غاما. مفتاح التحسين العميق الجديد يستخدم نموذج الذكاء الاصطناعي لنتائج أكثر جرأة.
- **ترميم الصورة** - أُعيدت كتابة اكتشاف الخدوش بترشيح Otsu بثماني زوايا. يعمل ملء LaMa الآن بالدقة الأصلية.
- **الصيغ الغريبة في كل مكان** - OCR وimage-to-PDF ومولّد الأيقونة المفضّلة والتركيب والدمج والتحويل إلى متجهات جميعها تفك ترميز HEIC وRAW وPSD الآن.
- **الضغط** - تشديد تحمّل الحجم المستهدف من 5% إلى 1%. الحجم المستهدف هو الوضع الافتراضي. إضافة أزرار زيادة/نقصان ومنتقي وحدة KB/MB.
- **تنظيف Sentry** - تصفية 644 حدثًا غير قابل للإجراء. الأخطاء الحقيقية تُعالَج الآن بشكل صحيح.
- **اكتشاف GPU** - تشخيصات أفضل للحاويات التي يوجد فيها CUDA لكن دون nvidia-smi.
- **وضع تعطيل المصادقة** - يُبذَر مستخدم مجهول في قاعدة البيانات بدور المسؤول. لم تعد مفاتيح API وخطوط الأنابيب وملفات المستخدم تتعطّل بسبب قيود المفاتيح الأجنبية.
- **أكثر من 2705 اختبارات جديدة** عبر الوحدات والتكامل والاختبار الشامل E2E.

### إصلاحات الأخطاء {#bug-fixes-3}

- لم يعد التكبير على CPU ينفد وقته على صناديق NAS والعتاد منخفض الطاقة.
- لم يعد شعار رمز QR يجعل المعاينة تختفي نهائيًا.
- إصلاح تجاوز القصّ للصور الطولية العالية.
- ملفات TIFF ذات قناة ألفا تفرض الآن إخراج PNG بشكل صحيح بدل إنتاج تلف.
- فك ترميز HDR/EXR يحوّل إلى 8 بت قبل CLAHE، ما يصلح إخفاقات فك الترميز.
- تحويل مخازن إدخال معالم الوجه إلى PNG قبل ذراع Python الجانبي، ما يصلح الانهيارات.
- البحث عن المكرّرات يتعامل مع الدفعات مختلطة الصيغ وأخطاء الشبكة.
- تحديث معاينة التجميل في الوقت الفعلي.
- أشرطة تقدّم لأداتي الدمج والتحويل إلى متجهات.
- التعامل مع SVGZ عبر أداة SVG إلى صورة نقطية.
- إصلاح أسماء الملفات غير اللاتينية عبر ترويسة X-File-Results المُرمَّزة بالنسبة المئوية.
